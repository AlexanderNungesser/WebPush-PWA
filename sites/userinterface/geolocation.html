Example 1: Use geolocation over function

Create a ask for geolocation useage with more user compfort options and more data security. 

<div id="geoloc_example1" swa="Geolocation"></div>
        

geoloc_example1_options = {
    onLocateFunctions: [
      function(position){
          alert("position recived. See javascript console for more information");
          console.log(position);
      }  
    ]
};

Example 2: Use geolocation by event

When a new geolocation is found, the swac_REQUESTORID_geolocation_newlocation event is fired.
It contains in the "detail" field the gps coordinates in field "position" and the translated address in "address".


document.addEventListener('swac_geoloc_example1_geolocation_newlocation',function(e) {
    console.log('New geolocation recived: ', e.detail.position);
    console.log('New geolocations address: ', e.detail.address);
});

Example 3: Use MagicMapper Plugin

MagicMapper is a device to improve location accuracy. It can be used with help of an API and the MagicMapperPlugin.
When activated an additional icon is shown in the geolocation bar. With click on that icon a dialog opens. Enter the address of the MagicMapper interface here. The plugin checks the connectivity and updates positions with locations from the MagicMapper.


var geoloc_example3_options = {
    plugins: new Map()
};
geoloc_example3_options.plugins.set('MagicMapperInterface', {
    id: 'MagicMapperInterface',
    active: true
});
