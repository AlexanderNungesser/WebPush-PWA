Example 1: Create filter for one datasource

Component declaration: 

<div id="datafilter_example1" swa="Datafilter FROM ../../data/visualisation/datafilter/example1.json"></div>
Component options: 

// Empty data for filteredDataSource (used by another component)
var filtered_data = [];

window['datafilter_example1_options'] = {
    originalDataRequestor: {
        fromName: 'exampledata_list.json'
    },
    filteredDataSource: 'filtered_data'
};
Useage of the filteredDataSource at another component: 

<div id="datafilter_example1_filtereddata" swa="Present FROM filtered_data"></div>
Datasets for component: 

[
    {
        "name": "Filter intval",
        "desc": "Filter datasets after intval",
        "attr": "intval",
        "type": "lt",
        "values": "2500"
    },
    {
        "name": "Filter stringval",
        "desc": "Filter datasets after stringval",
        "attr": "stringval",
        "type": "eq",
        "values": "string"
    }
]
